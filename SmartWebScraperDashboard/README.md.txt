# Smart Web Scraper Dashboard

A Python Flask web app that scrapes job listings, news headlines, and product prices, and displays them on a dashboard.

## Features
- Scrape data using BeautifulSoup
- Display results on a clean web UI
- Export options (coming soon)
- Filter/search (coming soon)

## Setup

```bash
git clone <repo_url>
cd SmartWebScraperDashboard
pip install -r requirements.txt
python run.py
